# 貢献ガイドライン

Legram プロジェクトへの貢献に興味を持っていただきありがとうございます！

## 開発環境のセットアップ

1. リポジトリをフォークしてクローンします。
2. 依存関係をインストールします:
   ```bash
   npm install
   ```
3. 環境変数を設定します (`.env.local`):
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. 開発サーバーを起動します:
   ```bash
   npm run dev
   ```

## プルリクエストの送信

- 小さな変更でも歓迎します。
- コードの変更内容を明確に記述してください。
- 既存のテストが通ることを確認してください（もしあれば）。
